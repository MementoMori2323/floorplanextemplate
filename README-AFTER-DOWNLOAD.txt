This template is mostly blank containing only a functional App Skeleton, Navbar, Supabase login, signup, 
forgot password and SmplrSpace Viewer along with security, control access and React Context for metadata.

Set repo to public for and download it with 

npx create-next-app@latest . --use-npm --example "https://github.com/MementoMori2323/floorplanextemplate"

Once downloaded make sure to the set the repo back to private.

In project update supabase api key, url and create the .env.local in the root with the following:

API_KEY=your_api_key_here
DATABASE_URL=your_database_url_here
